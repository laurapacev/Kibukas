# Kibukas
Kibukas is a chatting platform, which can be used as a way of communication between 

### Installation

Clone project 
```
git clone https://github.com/laurapacev/Kibukas.git
```

Build project dependencies (from project root)
```
cd kibukas
npm install
```

### Run project locally
From project root 
```
cd kibukas
npm start
```
